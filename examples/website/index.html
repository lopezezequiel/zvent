Zvent!

A Simple Router

Download

Features
  • Hash Routing
  • Form Routing
  • Methods GET, POST, PUT, PATCH, DELETE
  • Custom Methods
  • Query Strings
  • Parameter Capture
1. Basic Use

Use subscribe() to register a route.

2. Query Strings

Important: Zvent! does not decode querystrings. So it is better to use querystrings without encoding. If you want to use encoded querystrings anyway you can apply the decodeURI() function inside the callback.

3. Capture

Enclose in parenthesis what you want to capture.

4. Forms

Important: The form action have to start with '#'.

5. Custom Methods

You can create your own methods.

6. Trigger

Important: If you use the 'GET' method and do not pass the third parameter the data will be extracted from the querystring.

7. Shortcuts

POST(), GET(), PUT(), PATCH(), DELETE()
